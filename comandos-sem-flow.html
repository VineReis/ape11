-> git config --global user.name [SEU NOME] - Este comando define o nome de usuário do sistema Git.

-> git config --global user.email [SEU EMAIL] - Este comando define o email de usuário do sistema Git.

-> git init --bare [NOMEDOPROJETO].git - Este comando inicia um projeto com versionamento Git.

-> git clone [CAMINHODAPASTADOPROJETO] - Este comando clona o diretório padrão do projeto.

-> git remote add origin [CAMINHODAPASTADOPROJETO] - Este comando define a pasta de projeto como o diretório original do projeto.

-> git pull origin [BRANCH] - Este comando atualiza o branch do seu repositório com a ultima versão do branch do repositório de origem.

-> git add [ARQUIVO] - Este comando propoe mudanças em um arquivo (Adiciona ao index do seu diretório padrão).

-> git commit -m "COMENTARIO PARA MARCAR MODIFICAÇÃO" - Este comando confirma as mudanças que estão no index do seu diretório.

-> git push -u origin [BRANCH] - Este comando envia as ultimas modificações que estão no HEAD do seu diretório padrão, do BRANCH selecionado para o BRANCH do projeto de origem.

-> git checkout [BRANCH] - Este comando muda o branch ativo para o branch informado.

-> gitk - Este comando abre a interface gráfica padrão do git.
